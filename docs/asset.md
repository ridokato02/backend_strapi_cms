# Bảng Asset

- **Tên bảng:** `assets`
- **Loại:** `collectionType`
- **Tên hiển thị:** Asset

## Các trường

| Tên trường  | Kiểu dữ liệu | Chi tiết                                   |
| ----------- | ------------ | ------------------------------------------ |
| `filename`  | `string`     |                                            |
| `image_url` | `media`      | (multiple) allowedTypes: `["images", "files", "videos", "audios"]` |
| `type`      | `string`     |                                            |
| `size`      | `biginteger` |                                            |
